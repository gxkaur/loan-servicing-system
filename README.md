## What Each File Will Do

`loans.csv`

Stores the original loan account details.

`payments.csv`

Stores incoming payment transactions for a given month.

`loan.py`

Contains loan calculation logic like EMI and amortization basics.

`payment_processor.py`

Processes actual payments and updates balances.

`reporting.py`

Generates summary outputs like delinquent accounts and balances.

`main.py`

Runs the full workflow end to end.
